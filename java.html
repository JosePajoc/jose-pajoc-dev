Sistema administrativo de clientes, recursos y finanzas

El proyecto consiste en el desarrollo de una aplicación de escritorio con un menú de funciones administrativas clasificadas por funcionalidad, capaz de generar de forma dinámica distintas interfaces para que el usuario pueda administrar un negocio. Este cuenta con un sistema de carga de archivos CSV y la capacidad de generar reportes estadísticos con HTML y PDF. Para que el programa se pueda ejecutar es necesario instalar las librerías "jcommon-1.0.23", "jfreechart-1.0.19" y "itext5-itextpdf-5.5.12".
